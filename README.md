How to Play

Open the HTML file in your web browser.

Select the difficulty level.

You play as Black, the computer plays as White.

Click on a valid square to place your stone.

The game ends when no more valid moves are available.

The player with more stones on the board wins.

AI Logic (Overview)

Easy: Chooses a random valid move

Normal: Uses basic heuristics (e.g. number of flips)

Hard: Prioritizes strategic positions such as corners and edges

The AI is intentionally kept lightweight and readable rather than overly complex.
